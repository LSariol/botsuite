# Bot Suite (new name pending)
BotSuite is a scalable, multi-platform chat bot framework designed for creators, developers, and communities who need flexibility and reliability.
It supports multiple chat platforms, real-time message handling, and a fully modular architecture that makes new features easy to build and deploy.
